--- title: Cancer Survivors Network date: 2007-01-01 siteUrl: acscsn.org teaser: Home-grown to drupal data migration. teaserImage: csn-icon.png contentImage: csn-shot.png tags: - portfoliotype/not-for-profit - portfoliopay/paid --- The Cancer Survivors Network is an online support community run by the American Cancer Society. The site was formerly powered by a home-grown CMS, and it stored its data in a custom database schema to match. With the help of veteran consultant Moshe Weitzman, the CSN site is now being completely rebuilt, and is being powered by Drupal. Moshe brought me on board to help out with the task of migrating the data out of the old CSN database, and into the standard Drupal database schema. I was chosen for this task because of my work in developing the Import / Export API, as part of the Google Summer of Code 2006. I used the Import / Export API to help write a migration script for the user accounts and the forum threads on the site.
