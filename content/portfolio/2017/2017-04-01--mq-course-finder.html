--- title: MQ Course Finder date: 2017-04-01 siteUrl: courses.mq.edu.au teaser: Laravel back-end API development. teaserImage: mqcoursefinder-icon.png contentImage: mqcoursefinder-shot.png tags: - portfoliotype/business - portfoliopay/paid --- Macquarie University is one of Sydney's premier tertiary education institutions. Working freelance as part of the New Ground team, I helped to build the University's new Course Finder site. I was one of the lead API developers working on the app's Laravel PHP back-end. The front-end site (slash mobile app) was built using React. The app allows data from the existing MQ handbook database to be imported, for every new year of course offerings. It exposes course and related info via a collection of RESTful APIs.
